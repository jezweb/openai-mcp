This page is designed to display and test DALL-E generated images. To use it:

  2. Copy the image URL from the console output
  3. Open your browser's console (F12 or right-click > Inspect > Console)
  4. Run the following code in the console, replacing the URL with your image URL:
const imageUrl = "https://your-dalle-image-url-here.png";
document.getElementById('image-display').innerHTML = `<img src="${imageUrl}" alt="DALL-E Generated Image">`;

DALL-E API Options

The DALL-E API supports various options to customize the image generation process. Here's a comprehensive list of available options:

Parameter Description Values
model The DALL-E model to use dall-e-2, dall-e-3
prompt Text description of the desired image String (max 1000 characters)
n Number of images to generate 1-10 (DALL-E 2), 1 (DALL-E 3)
size Size of the generated image 256x256, 512x512, 1024x1024 (all models)
1792x1024, 1024x1792 (DALL-E 3 only)
quality Quality of the generated image standard, hd (DALL-E 3 only)
style Style of the generated image vivid, natural (DALL-E 3 only)
response_format Format of the response url, b64_json

Example Configurations

Default

{
  prompt: "A cat wearing a hat"
}

Uses all default settings: DALL-E 3, 1024x1024, standard quality, vivid style.

HD Quality

{
  prompt: "A scenic mountain landscape at sunset",
  quality: "hd",
  size: "1024x1024"
}

Uses HD quality for more detailed images (DALL-E 3 only).

Natural Style

{
  prompt: "A futuristic city with flying cars",
  style: "natural"
}

Uses natural style for more photorealistic images (DALL-E 3 only).

Wide Format

{
  prompt: "A panoramic view of a beach at sunrise",
  size: "1792x1024"
}

Uses wide format for landscape-oriented images (DALL-E 3 only).

Tall Format

{
  prompt: "A tall skyscraper reaching into the clouds",
  size: "1024x1792"
}

Uses tall format for portrait-oriented images (DALL-E 3 only).
